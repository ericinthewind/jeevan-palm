# Jeevan Palm

Première version du site vitrine Astro.

## Ouvrir dans Cursor

1. Décompresser le dossier.
2. Ouvrir le dossier `jeevan-palm` dans Cursor.
3. Lancer :

```bash
npm install
npm run dev
```

## Build Cloudflare Pages

- Build command: `npm run build`
- Output directory: `dist`

## À personnaliser

- Remplacer `https://jeevanpalm.com` dans `astro.config.mjs` par le domaine final.
- Vérifier et compléter les textes et équipements.
- Ajouter la version anglaise dans une prochaine itération.
- Les photos optimisées sont dans `public/images`.
